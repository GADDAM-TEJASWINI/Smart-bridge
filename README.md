# INTELLISQL
IntelliSQL is an AI-powered Natural Language to SQL (NL2SQL) web application designed to simplify database interaction through intelligent query generation. The system enables users to retrieve structured data by asking questions in plain English, eliminating the need to manually write complex SQL statements.

Built using Streamlit for the front-end interface, SQLite as the relational database engine, and Google’s Gemini Large Language Model for intelligent query conversion, IntelliSQL seamlessly translates natural language inputs into syntactically accurate and executable SQL queries. The generated queries are automatically executed against the database, and results are displayed in real time within a user-friendly interface.

The project demonstrates the practical integration of Generative AI into data management workflows. By bridging the gap between human language and structured database systems, IntelliSQL enhances accessibility, reduces technical barriers, and minimizes query-writing errors.

IntelliSQL serves as a proof-of-concept for AI-driven database automation and highlights the potential of Large Language Models in transforming traditional data retrieval processes into more intuitive and efficient experiences.
